# Icônes pour l'extension

Pour ajouter des icônes à l'extension, créez 3 fichiers PNG :
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

Puis décommentez les sections `icons` et `default_icon` dans `manifest.json`.

Pour l'instant, l'extension fonctionne sans icônes personnalisées.
